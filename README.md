## 🚀 About the Project
This project is a simple, interactive web-based implementation of the classic hand game. It allows users to play a quick and fun game directly from their web browser, showcasing fundamental web development skills through DOM manipulation and basic game logic.

## 🕹️ How the Game Works
Rock-Paper-Scissors is a simple hand game usually played against an opponent. In this web version, you are playing against the computer.

The rules are straightforward:
* 💎 **Rock** crushes **Scissors** ✂️ (Rock wins)
* ✂️ **Scissors** cuts **Paper** 📄 (Scissors wins)
* 📄 **Paper** covers **Rock** 💎 (Paper wins)

**To play:**
1. Click on the button/icon corresponding to your choice (Rock, Paper, or Scissors).
2. The computer will randomly generate its own choice.
3. The game will compare the two choices and instantly display the winner of the round. 🏆
4. If both you and the computer choose the same shape, the round is a tie! 🤝

## 🛠️ Technologies Used
This project was built using standard web technologies:
* 🟨 **JavaScript (47.0%):** Handles the game logic, random choice generation for the computer, and updates the user interface based on the results.
* 🟧 **HTML (18.5%):** Provides the basic structure and layout of the webpage.
* 🟦 **CSS (34.5%):** Styles the elements to make the game visually appealing.
